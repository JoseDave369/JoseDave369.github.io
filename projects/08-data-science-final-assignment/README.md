## 📋 Project Overview

Capstone project of the **IBM Data Science Professional Certificate** on Coursera. Applied the complete data science lifecycle from problem definition through analysis, modelling, and communication of results.

## 🔬 Data Science Workflow

```
1. Problem Definition      → Business question framing
2. Data Collection         → Dataset acquisition and loading
3. Exploratory Data Analysis (EDA) → Statistics, distributions, correlations
4. Data Wrangling          → Cleaning, missing values, feature engineering
5. Machine Learning        → Model selection, training, tuning
6. Evaluation              → Accuracy metrics, cross-validation
7. Storytelling            → Results communication in Jupyter Notebook
```
