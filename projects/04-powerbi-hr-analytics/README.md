## 📋 Project Overview

HR Analytics dashboard providing HR managers and executives with deep insights into workforce composition, attrition risk, salary distribution, and departmental performance metrics.

## 📊 Key Metrics Tracked

| Category | Metrics |
|---|---|
| Workforce | Headcount, new hires, active employees |
| Attrition | Turnover rate, attrition by department/age/role |
| Performance | Ratings distribution, promotion eligibility |
| Compensation | Salary bands, average salary by department |
| Diversity | Gender distribution, age cohort analysis |

## 🔑 Dashboard Features

- **Attrition Analysis:** Identifies flight-risk segments using demographic filters
- **Department Breakdown:** Compare performance and headcount across business units
- **Tenure Distribution:** Visualises employee loyalty and churn patterns
- **Salary Benchmarking:** Compares compensation against performance scores
- **Interactive Filtering:** Slicers for department, job role, gender, education level
